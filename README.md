# Eswar's Portfolio

Personal portfolio website of **Guru Eswar Sainath Reddy Kummithi**, a Bengaluru-based full-stack software developer with experience in React, Node.js, Rust, and AWS.

Live at: [eswar-k.netlify.app](https://eswar-k.netlify.app)

## Tech Stack

- React + Vite
- Tailwind CSS
- React Router
- Deployed on Netlify

## Sections

- **Intro** - Overview, resume download, and contact link
- **Journey** - Professional experience timeline
- **Projects** - Work across frontend, full-stack, and systems engineering
- **Get in Touch** - Contact via email and social links

## Running Locally

```bash
npm install
npm run dev
```
